# Cypress JavaScript BDD Framework

BDD test automation framework built with Cypress, Cucumber, and JavaScript. Covers API and UI test scenarios with Allure and Mochawesome reports, runs in Docker, integrates with GitHub Actions CI, and enforces code quality via ESLint and Prettier.
